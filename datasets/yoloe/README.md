# YOLOE Dataset Resources

This directory contains dataset-related resources used by the YOLOE sample.

## Files

- `yoloe_seg_pf_classes.names` — Class names file for YOLOE-11 Seg Prompt-Free model, containing 4585 class names for open-vocabulary instance segmentation.

## Usage

The class names file is referenced by the YOLOE Python runtime for visualization of detected instances:

```bash
python3 main.py --label-file ../../../../../datasets/yoloe/yoloe_seg_pf_classes.names
```

## Notes

- The 4585 classes cover a broad vocabulary for open-vocabulary detection and segmentation tasks.
- This file is generated by the ONNX export script (`conversion/onnx_export/export_yoloe11seg_bpu.py`) during model conversion.
