# YOLOE Model Files

This directory contains the download script for the `yoloe` sample model files.

- Target platform: `RDK X5`
- Model format: `.bin`
- Default model: `yoloe-11s-seg-pf_bayese_640x640_nv12.bin`

## Download

Run the download script to fetch the model:

```bash
bash download_model.sh
```

This downloads the model file to the current directory.

## Available Models

### Bayes-e (RDK X5 & RDK X5 Module)

#### bin - nv12

YOLOE-11s-Seg-PF

```bash
wget https://archive.d-robotics.cc/downloads/rdk_model_zoo/rdk_x5/AAA_RDK_YOLO/yoloe-11s-seg-pf_bayese_640x640_nv12.bin
```
