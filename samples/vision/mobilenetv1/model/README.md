# Model Download

Run the download script from this directory:

```bash
bash download_model.sh s100
```

The model is downloaded to:

```text
model/s100/mobilenetv1_224x224_nv12.hbm
```

## Published Model

| File | Platform | Input |
| --- | --- | --- |
| `mobilenetv1_224x224_nv12.hbm` | S100 | NV12 (Y + UV) |

Download source:

```text
https://archive.d-robotics.cc/downloads/rdk_model_zoo/rdk_s100/MobileNet/mobilenetv1_224x224_nv12.hbm
```

No S100P-specific public HBM URL is available in the checked download paths.
