# 模型下载

在本目录执行下载脚本：

```bash
bash download_model.sh s100
```

模型会下载到：

```text
model/s100/mobilenetv1_224x224_nv12.hbm
```

## 已发布模型

| 文件 | 平台 | 输入 |
| --- | --- | --- |
| `mobilenetv1_224x224_nv12.hbm` | S100 | NV12 (Y + UV) |

下载地址：

```text
https://archive.d-robotics.cc/downloads/rdk_model_zoo/rdk_s100/MobileNet/mobilenetv1_224x224_nv12.hbm
```
