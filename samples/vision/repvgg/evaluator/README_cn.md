# 模型评测

本目录提供 RepVGG sample 的 benchmark 说明和验证参考。

## 支持模型

| 模型 | 输入尺寸 | 类别数 |
| --- | --- | --- |
| RepVGG_A0 | 224x224 | 1000 |
| RepVGG_A1 | 224x224 | 1000 |
| RepVGG_A2 | 224x224 | 1000 |
| RepVGG_B0 | 224x224 | 1000 |
| RepVGG_B1g2 | 224x224 | 1000 |
| RepVGG_B1g4 | 224x224 | 1000 |

## 测试环境

- 平台：`RDK X5`
- 运行后端：`hbm_runtime`
- 模型格式：`.bin`
- CPU：8xA55@1.8GHz，全核 Performance 调度
- BPU：1xBayes-e@1GHz，10TOPS 等效 INT8 算力

## 指标说明

- 浮点 Top-1 为量化前 ONNX 浮点模型精度。
- 量化 Top-1 为量化部署模型精度。
- 单线程延迟为单帧、单线程、单 BPU 核心推理延迟。
- 多线程延迟为多线程提交任务时的延迟。
- FPS 为 `RDK X5` 上多线程吞吐性能。
