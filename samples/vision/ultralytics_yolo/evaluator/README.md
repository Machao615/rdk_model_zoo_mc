# Evaluator

This directory contains tools and documentation for evaluating Ultralytics YOLO model accuracy and performance on RDK S100/S100P.

Evaluation details are pending and will be added in a future update.

## Evaluation Tasks

- **Detection**: COCO mAP evaluation (AP50, AP50:95)
- **Segmentation**: COCO instance segmentation mAP
- **Pose**: COCO keypoint AP (OKS-based)
- **Classification**: ImageNet Top-1 / Top-5 accuracy
